# dib

CLI build/release tool for some misc. tasks

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dib'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install dib

Or install the gem:

```bash
gem install pkg/dib-0.0.1.gem
```

## Usage

run `./bin/dib` for usage instructions
